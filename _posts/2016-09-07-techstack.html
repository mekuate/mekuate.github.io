Pour développer un site web de qualité, il est important de bien maîtriser des technologies modernes de création de site internet. Cet article présente les technologies utilisées, ceux donc je maitrise parfaitement, pour le développement des sites web:

Drupal 7/8

Le système de gestion de contenu Drupal est le logiciel utilisé lorsqu’il s’agit de créer un site web avec une forte dynamique des informations, particulièrement les sites web d’actualités, ou donc les contenus sont structurés par catégories et types de contenus complexes des organisations. J’ai une préférence à Drupal par rapport à JOMLA et Wordpress parceque DRUPAL est modulable, permet de réaliser des sites web, des plus simples au plus complexes; est écrit en PHP et donc j’ai une bonne expérience, ce qui me permet de programmer des modules Drupal.

La plateforme Acquia Dev Desktop est utilisé pour deployer en un clic en local l'environnement de développement sous Drupal. Acquia Dev Desktop intègre Git, Drush

Sass et Compass

SASS une extension du CSS et permet d’attendre les possibilités syntaxiques du CSS. par exemple, elle permet de définir les variables dans les feuilles de style CSS. d’oú un avantage de maintenance et de réutilisabilité en utilisant SASS. Il existe aussi LESS, comme alternative à SASS mais j’ai préféré apprendre SASS par convenance.

Compass est un préprocesseur de CSS utilisé pour convertir les fichiers SASS (.scss) en CSS.

Twitter Bootstrap

Twitter Bootstrap est une librairie avancée utilisée pour le design des sites web. Il permet de booster le développement du design des sites web, et est très bien intégrer à Drupal grâce à des multiples modules particulierement le starter kit de twitter bootstrap pour drupal 7 et 8 (thème de base), le module views bootstrap, Panel bootstrap et panelizer.

Jquery

JQuery est la librairie Javascript préférée de la communauté DRUPAL, bien que l’on puisse utiliser d’autres librairies comme Reactjs, BlackBonejs, angularjs. j’utilise au mieux Jquery pour des animations sur les sites web et bien d’autres.

git/Github

Le controle de version est le coeur du processus de développement des systèmes informatiques. L'usage des lignes de commandes git pour la gestion de versionning du site web procure une grande flexibilité du processus itératif et agile de mise en oeuvre de la plateforme.

Github est une interface web de gestion des dépôts des codes sources par l'intermédiaire de git. Avec un compte Github, la gestion des versions des projects informatiques est aisée. De Github Desktop , aux wikis et Github pages , les outils de Github sont très utiles.

Nginx

La meilleure alternative au serveur web apache à mon avis, car conçu pour gérer plusieurs requêtes de page web en même temps (concurrentiels ou pas) sur le serveur. Nginx est tout aussi gratuit d'utilisation que Apache et bien plus facile à configurer les virtuals hosts pour héberger plusieurs sites sur un même serveur disposant d'une seule adresse IP.

Jekyll

Jekyll est un générateur de site web statique. Les sites web statiques sont idéales pour les sites dites "Brochures" ou les blogs. Jekyll vient à point pour reconsidérer les sites web statiques au détriment des sites dynamiques (utilisant une base de données).

D'ailleurs ce site web est construit entièrement avec Jekyll.
